## 2. 用 lambda 构造过程

其中有这样一个例子：

```lisp
(+  (let    ((x 3))
        (+ x (* x 10)))
    x)
```

如果 `x` 的值为 5，那么 + 的第二个操作数值仍旧为 5，哪怕在第一个操作数中，`let` 将 `x` 的值绑定为 3。

即使在 `let` 约束局部变量时用到了先前约束的变量：

```lisp
(let    ((x 3)
        (y (+ x 2)))
    (+ x y))
```

在这里，用于赋给 `y` 的值 `x`，也不是上面绑定的 `x`，而是外部的 `x`。
